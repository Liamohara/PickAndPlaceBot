# PickAndPlaceBot
### Objective
A 3 DOF Planar pick-and-place bot which picks up a disc from a marked area, flips it and places it in a box.


### Hardware
- IGUS Apiro Robotics Kit
- ESP32-S3-DEV-KIT-N8R8
- 3x DM556T Stepper Drivers
- 3x Drylin NEMA17 Stepper Motors
- Servo Motor
- DFROBOT DR1154 ESP32-S3 Camera Module
- Custome grabber, camera mount and "slide" 3D prints
